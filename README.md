# study-pydantic

本仓库用于学习 **Pydantic** 以及 **PydanticAI**。

## 安装依赖

建议使用 Python 3.10+。

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 最小示例（PydanticAI）

见 `examples/minimal_pydantic_ai.py`。

```bash
source .venv/bin/activate
python examples/minimal_pydantic_ai.py
```
